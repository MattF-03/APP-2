## Description :
Programme Python qui permet de classer des avions en approche de d'un aéroport en fonction de critères définis.

## Installation / Utilisation :
Lorsqu'on lance le programme, la simulation se lance et il fait attérir les avions selon des policies et situations différentes. 

## Structure du Projet :
Il y a 4 codes extérieur qui permettent de faire fonctionner le main. On a un code de tri, générateur, simulation et policies. 
Notre main réutilise chacun de ces codes pour faire la simulation final, ainsi que la comparaison entre différent type de tri.
