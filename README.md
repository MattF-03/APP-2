## Description :
Programme Python qui permet de classer des avions en approche de d'un aéroport en fonction de critères définis.

## Installation / Utilisation :
Lorsqu'on lance le programme, la simulation se lance et fait atterrir les avions selon des politiques et situations différentes.
## Structure du Projet :
Il y a 4 codes extérieurs qui permettent de faire fonctionner le main. On a un code de tri, un générateur, une simulation et des politiques (policies). Notre main réutilise chacun de ces codes pour faire la simulation finale, ainsi que la comparaison entre différents types de tris.
